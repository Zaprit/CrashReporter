Welcome to the admin page

From here you can see and manage all the reports that come in from users

If users are spamming reports then use the ban button to prevent that IP address from making more reports

If sufficient users report the same issue then it should be flagged as an issue on the lighthouse Github

Anything that doesn't have anything to do with the lighthouse code itself though shouldn't be reported as an issue on github

Once reviewed, leave a comment and either resolve it or close it depending on wether or not it was a bug or user error
